Папка для подготовки MIB файлов

-------------------------------- Содержимое ----------------------------------------

KO5_002.mib	- стандартный mib файл для mib браузера (клиента)
KO5_002.mcp.mib - mib файл в формате микрочип

mib2bib.jar     - утилита, преобразующая микрочиповский mib файл в бинарный (*.bib)
CreateBIB.bat   - батник для преобразования KO5_002.mcp.mib в snmp.bib


----------------------------- Порядок работы ---------------------------------------

*   Вручную внести изменения в KO5_002.mcp.mib согласно AN870 так, чтобы были прописаны
    все листья, содержащиеся в оригинальном  KO5_002.mib файле

*   Запустить CreateBIB.bat   
    Будут сформированы файлы snmp.bib и mib.h

*   snmp.h кинуть в папку WebPages2;   mib.h - в корневую папку проекта

*   Запустить утилиту MPFS2 из папки Utilites и сформировать новый образ файловой системы
    для С18/32 Image, c Image name = MPFSImg2.c

*   Перекомпилировать проект
